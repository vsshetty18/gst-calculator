# gst-calculator

A free online GST Calculator built with **Next.js 15** for the Digital Heroes trial task.

## 👤 Author

**V S VIGHNESH**  
📧 your-email@gmail.com

## 🚀 Features

- Calculate GST at 5%, 12%, 18%, and 28% rates
- Instantly see Original Amount, GST Amount, and Total Amount
- Input validation (empty, negative, non-numeric)
- Fully responsive — mobile & desktop friendly
- Premium blue-gradient modern UI
- Reset button to clear fields

## 🛠️ Tech Stack

- Next.js 15 (App Router)
- React 19
- Plain CSS (no Tailwind)
- Vercel-ready deployment
